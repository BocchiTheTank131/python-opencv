## How can I update the code in GitHub when i finished the code?

> Adding file
- Add all of the file when you already modified.
```
git add .
```
- Add a single file.
```
git add week01.py
```

> Commenting file
- Comment the code.
```
git commit -m "comment"
```

> Update the code
- pushing code.
```
git push -u origin main
```
